# PetWays Legal Site

Static legal website for PetWays, operated by **S.A.M LLC** and doing business as **PetWays**.

Repository target:

<https://github.com/suhasdaniel325-commits/petways-legal>

## Pages

- `index.html` - legal site landing page
- `privacy-policy.html` - Privacy Policy for Google Play and public app users
- `terms-of-service.html` - Terms of Service for MVP launch
- `contact.html` - support and privacy contact page

## Publishing With GitHub Pages

1. Push these files to the `main` branch of the `petways-legal` repository.
2. In GitHub, open **Settings > Pages**.
3. Set the source to deploy from the `main` branch root.
4. Use the published Privacy Policy URL in Google Play Console.

## Before Publishing

- Replace `petwayssupport@gmail.com` if a different final support email is chosen.
- Review the policy text with legal counsel if needed.
- Confirm the effective date before submitting to Google Play.

No Firebase keys, app source code, signing files, or secrets belong in this repository.
